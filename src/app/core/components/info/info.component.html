About Frescolist

Author

Hi! I'm Francesco, a software developer passionate about music and automation. You can find me on GitHub.

Project

Frescolist is an open-source project currently in alpha that helps you create and edit Spotify playlists using text files. The main goal is to make it easy to create playlists from AI-generated song lists or any text-based source.

Privacy

Frescolist runs entirely in your browser. No data is ever sent to or stored on external servers - everything stays on your device and is handled directly with Spotify's API.

Project Status

Alpha

This project is under active development. Features and UI might change.
